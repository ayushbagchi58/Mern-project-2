The project contains a React client and a Node.js/Express server, connected through REST APIs with MongoDB as the database. It includes features like create product,view single product details, data listing, filtering, updating, and deletion.

## Install

```sh
npm install && cd client && npm install && cd ../server && npm install
```

## Usage

```sh
cd server && npm start & cd ../client && npm run dev
```
